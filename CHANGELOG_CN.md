## [0.1.1] - 2026-02-10

### 🎉 首次发布

这是 **R2Droid** 的首个公开版本 - 一个基于 Radare2 逆向工程框架的现代化 Android 原生 GUI 工具。使用 Kotlin 和 Jetpack Compose (Material3) 构建，R2Droid 为移动设备带来了强大的二进制分析能力。

### ✨ 核心功能

#### 📱 项目管理
- **保存与恢复项目**：持久化项目状态及 r2 分析缓存
- **项目索引**：管理多个项目，包含元数据（文件大小、分析时间、最后打开时间）
- **自定义项目目录**：配置项目保存位置
- **快速访问**：主屏幕显示最近项目

#### 🔍 分析视图

**十六进制编辑器**
- 针对大文件的高性能虚拟化列表渲染
- 使用 `wx`、`w`、`wa` 命令进行实时 hex/ASCII 编辑
- 基于列的选择和高亮显示
- 适配移动端的自定义 hex 键盘
- 采用 LRU 缓存和分块加载实现最优内存使用

**反汇编视图**
- 支持无限滚动的虚拟化渲染
- 操作码、寄存器、立即数的语法高亮
- 控制流跳转箭头可视化
- 显示函数边界和注释
- 导航历史记录（前进/后退）
- 基于地址的搜索和跳转

**反编译器**
- 使用 r2 分析插件生成伪代码
- 语法高亮显示
- 地址可点击快速导航
- 支持多种反编译后端

#### 📊 二进制信息

- **概览**：架构、位数、操作系统、入口点、文件大小
- **段（Sections）**：加载地址、大小、权限、熵值
- **符号（Symbols）**：函数、变量、导入、导出
- **字符串（Strings）**：可搜索的字符串表及地址
- **函数列表**：完整的函数列表及分析详情
- **导入/导出**：外部依赖和对外暴露的 API
- **交叉引用（Xrefs）**：代码和数据之间的引用追踪

#### 🔎 搜索功能
- 搜索字节、字符串和指令
- 按类型过滤结果
- 快速导航到匹配项
- 搜索结果上下文显示

#### 💻 终端集成
- 使用 termux-view 的全功能终端
- 通过 r2pipe 执行原生 r2 命令
- 命令历史记录
- 自定义命令快捷方式
- 实时输出流

#### ⚙️ 设置与自定义

- **主题**：支持深色/浅色模式，采用 Material3 设计
- **语言**：支持英文和中文（简体）
- **自定义字体**：为代码视图加载外部字体文件
- **R2 配置**：内置 `.radare2rc` 编辑器用于 r2 自定义
- **分析选项**：配置分析深度（`aa`、`aaa`、`aaaa`）

### 🛠️ 技术亮点

- **架构**：采用 MVVM 模式，使用 Hilt 依赖注入
- **UI 框架**：Jetpack Compose 与 Material3 组件
- **性能**：hex 和反汇编视图的虚拟化列表可处理任意大小的文件
- **内存管理**：基于分块的加载机制配合 LRU 缓存
- **R2 集成**：适配 Android 的原生 r2pipe 实现
- **Termux 集成**：终端模拟器后端和视图组件

### 📦 包含组件

- **Radare2**：预编译的 r2 二进制文件和库（v5.9.x）
- **终端模拟器**：基于 Termux 终端组件
- **R2 库**：原生共享库（libc++、libz）

### 🐛 已知问题

- 首次安装需要解压约 70MB 的 r2 资源
- 部分 r2 命令在移动端可能功能受限
- 图形视图（VV）尚未实现
- ESIL 模拟 UI 待开发

### 🔜 即将推出

- 可视化调用图（VV）查看器
- 增强的终端快捷键支持
- ESIL 调试器界面
- 改进的分析进度反馈
- 更多 r2 插件支持
