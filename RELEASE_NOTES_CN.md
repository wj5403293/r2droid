# R2Droid v0.1.1 发布说明

## 🎉 首次发布

**R2Droid** 是一个基于 [Radare2](https://github.com/radareorg/radare2) 逆向工程框架的现代化 Android 原生 GUI 工具。使用 Kotlin 和 Jetpack Compose (Material3) 构建，为移动设备带来了强大的二进制分析能力。

---

## ✨ 主要功能

### 📱 项目管理
- 保存和恢复完整的分析会话及 r2 缓存
- 项目元数据追踪（文件大小、分析时间、最后打开时间）
- 支持自定义项目目录
- 最近项目快速访问

### 🔍 高级分析工具

#### 十六进制编辑器
- ⚡ 针对任意大小文件的高性能虚拟化渲染
- ✏️ 使用 `wx`、`w`、`wa` 命令实时编辑
- 🎯 基于列的选择和高亮
- ⌨️ 适配移动端的自定义 hex 键盘
- 💾 智能 LRU 缓存与分块加载

#### 反汇编视图
- 📜 支持虚拟化的无限滚动
- 🎨 语法高亮（操作码、寄存器、立即数）
- ➡️ 控制流跳转箭头可视化
- 📍 函数边界和注释显示
- 🔄 导航历史（前进/后退）
- 🔎 基于地址的搜索和跳转

#### 反编译器
- 📝 通过 r2 插件生成伪代码
- 🎨 语法高亮
- 🔗 地址可点击导航
- 🔧 支持多种反编译后端

### 📊 二进制信息视图
- **概览**：架构、位数、操作系统、入口点、文件大小
- **段**：地址、大小、权限、熵值
- **符号**：函数、变量、导入、导出
- **字符串**：可搜索的字符串表
- **函数**：完整的函数列表及详情
- **交叉引用**：引用关系追踪

### 🔎 搜索系统
- 搜索字节、字符串和指令
- 按类型过滤
- 快速导航到结果
- 上下文显示

### 💻 集成终端
- 基于 termux-view 的完整 r2pipe 终端
- 执行原生 r2 命令
- 命令历史
- 自定义快捷方式
- 实时输出

### ⚙️ 自定义选项
- 🌓 深色/浅色主题（Material3）
- 🌍 支持英文和中文
- 🔤 自定义字体加载
- ⚙️ 内置 `.radare2rc` 编辑器
- 🔬 可配置分析深度（`aa`、`aaa`、`aaaa`）

---

## 🛠️ 技术细节

- **架构**：MVVM 模式 + Hilt 依赖注入
- **UI**：Jetpack Compose + Material3
- **性能**：虚拟化列表处理无限大小文件
- **内存**：基于分块的加载机制配合 LRU 缓存
- **集成**：适配 Android 的原生 r2pipe + Termux 组件
- **包含**：预编译 Radare2 v5.9.x（约 70MB）

---

## 📥 安装说明

1. 从 [Releases 页面](https://github.com/wsdx233/r2droid/releases)下载 APK
2. 在 Android 设备上安装（最低 SDK 24）
3. 授予存储权限
4. 首次启动将解压 r2 资源（约 70MB）
5. 打开二进制文件开始分析
